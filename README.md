# kunkun-game

## 项目背景
基于pyqt5和pygame的一种技术尝试,
使用OpenCV进行图像处理
使用pyqt的QLabel展示视频画面
使用pygame进行游戏逻辑判断

## 运行环境
Ubuntu 20.04
python 3.8
